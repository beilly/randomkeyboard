# randomkeyboard
基于PopWindow封装的自定义键盘，支持随机键值

### 主要有一下优点：

1、纯数字的自定义键盘；

2、使用助手类的方式，可以使用一个对象管理多个输入框；

3、不依赖于页面布局，使用内部使用Popwindow的方式实现键盘弹出，更加模块化。

4、建议使用源码的Module的方式引入，方便修改样式；
